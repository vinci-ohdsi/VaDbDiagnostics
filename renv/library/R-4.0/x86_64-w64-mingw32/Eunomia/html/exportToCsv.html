exportToCsv {Eunomia}R Documentation

Extract the Eunomia database to CSV files

Description

Extract the Eunomia database to CSV files

Usage

exportToCsv(
  outputFolder = file.path(getwd(), "csv"),
  connectionDetails = getEunomiaConnectionDetails()
)

Arguments

outputFolder

A folder where the CSV files will be written.

connectionDetails

Connection details for the Eunomia database. Defaults to a fresh Eunomia database.

Examples


# For this example we'll create a temp folder:
folder <- tempfile()
dir.create(folder)

exportToCsv(folder)

list.files(folder)
