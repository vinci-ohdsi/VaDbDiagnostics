Description

Stop the cluster

Usage

stopCluster(cluster)

Arguments

cluster

The cluster to stop

Examples

fun <- function(x) {
  return (x^2)
}

cluster <- makeCluster(numberOfThreads = 3)
clusterApply(cluster, 1:10, fun)
stopCluster(cluster)
