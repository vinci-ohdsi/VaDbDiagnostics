Trie Getters

Description

"Getters" for the data stored in a trie object. get_keys gets the keys, get_values gets the values.

Usage

get_keys(trie)

get_values(trie)

Arguments

trie

A trie object, created with trie.

Value

An atomic vector of keys or values stored in the trie.
